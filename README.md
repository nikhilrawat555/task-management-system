 # Task Management App

A simple and responsive Task Management Web Application for creating, updating, and tracking tasks.

## Live Demo
[Task Management App](https://github.com/nikhilrawat555/task-management-system)

## Features
- Add tasks with title and priority
- Set Priority — High, Medium, Low
- Mark tasks as Completed
- Delete tasks
- Filter tasks — All / Pending / Completed
- Data persists using LocalStorage
- Responsive design for web and mobile screens

## Tech Stack
- HTML5
- CSS3
- JavaScript (Vanilla)
- LocalStorage API
